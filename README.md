## Projects

<br/>

**[Sample Pack Assembler](https://github.com/strykey/Sample-Pack-Assembler)** &nbsp; — &nbsp; A desktop app for producers who are tired of having a folder called "samples_FINAL_v3_USE_THIS". Scan a messy folder, auto-categorize everything, rename in bulk, export a clean pack. One click. Done.

<br/>

**[Pixora](https://github.com/strykey/Pixora)** &nbsp; — &nbsp; Drop an image, get pixel art back. Perceptual color quantization, Floyd-Steinberg dithering, edge enhancement. The kind of thing that should have existed already and somehow didn't.

<br/>

**[IRIS](https://github.com/strykey/IRIS)** &nbsp; — &nbsp; A multithreaded reconnaissance tool that scans random public IPs and surfaces exposed devices in real time. Cameras, routers, admin panels, forgotten websites. The internet is bigger and stranger than what Google shows you.

<br/>

**[OrbsHacker](https://github.com/strykey/orbshacker)** &nbsp; — &nbsp; Automatically creates fake game processes for Discord Orb quests without installing the actual games. Because nobody has time to download 500GB just to click a button on Discord.
