I build things. Most of what I make starts as a personal problem I got tired of working around. At some point it felt wrong to keep it private, so I put it out there. Some of it is useful for producers, some of it is useful for people who are curious about how the internet actually works. Some of it is just because I thought it would be fun to make.

I stay anonymous. The work speaks for itself.

## Projects

<a href="https://github.com/strykey/Sample-Pack-Assembler">
<img align="right" width="45%" src="https://github-readme-stats.vercel.app/api/pin/?username=strykey&repo=Sample-Pack-Assembler&bg_color=0a0a0a&border_color=2a2a2a&title_color=c8b89a&text_color=e8e0d0&icon_color=c8b89a&hide_border=false" />

**Sample Pack Assembler**

A desktop app for producers who are tired of folders called `samples_FINAL_v3_USE_THIS`. Scan a messy directory, let it auto-categorize everything across 14 types, rename in bulk with a template system, export a clean structured pack. One click. Done. Built with pywebview, five distinct themes, audio preview on hover.

**Pixora**

Drop an image, get pixel art back. Perceptual color quantization, Floyd-Steinberg dithering, smart auto pixel sizing and edge enhancement. The kind of converter that should have existed already and somehow didn't. No account, no upload, no waiting.

<a href="https://github.com/strykey/IRIS">
<img align="right" width="45%" src="https://github-readme-stats.vercel.app/api/pin/?username=strykey&repo=IRIS&bg_color=0a0a0a&border_color=2a2a2a&title_color=c8b89a&text_color=e8e0d0&icon_color=c8b89a&hide_border=false" />

**IRIS**

A multithreaded reconnaissance tool that scans random public IPs and surfaces exposed devices in real time. Cameras, routers, admin panels, live streams, forgotten websites. The internet is bigger and stranger than what Google shows you. IRIS looks at scale.

<br clear="right"/>

<br/>

<a href="https://github.com/strykey/orbshacker">
<img align="right" width="45%" src="https://github-readme-stats.vercel.app/api/pin/?username=strykey&repo=orbshacker&bg_color=0a0a0a&border_color=2a2a2a&title_color=c8b89a&text_color=e8e0d0&icon_color=c8b89a&hide_border=false" />
</a>

**OrbsHacker**

Fakes game processes for Discord Orb quests without installing the actual games. Because nobody has time to download 500GB just to click a button on Discord. It works, it's stupid, and I have zero regrets building it.
