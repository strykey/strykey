<h2 align="center">What I have built so far</h2>

###

<div align="center">

[![Sample Pack Assembler](https://img.shields.io/badge/Sample%20Pack%20Assembler-View%20Project-c8b89a?style=for-the-badge&logo=python&logoColor=0a0a0a&labelColor=1a1a1a)](https://github.com/strykey/Sample-Pack-Assembler)

A desktop app for producers tired of folders named `samples_FINAL_v3_USE_THIS`.<br/>
Scan a messy directory, auto-categorize across 14 types, rename with a template system, export a clean pack.<br/>
Five visual themes, audio preview on hover, one Python file, one dependency.<br/>

<br/>

[![Pixora](https://img.shields.io/badge/Pixora-View%20Project-c8b89a?style=for-the-badge&logo=python&logoColor=0a0a0a&labelColor=1a1a1a)](https://github.com/strykey/Pixora)

Convert any image into authentic pixel art.<br/>
Smart auto pixel sizing, perceptual color quantization, Floyd-Steinberg dithering and edge enhancement.<br/>
Drop your image, hit convert, done. No account, no upload, no waiting.<br/>

<br/>

[![IRIS](https://img.shields.io/badge/IRIS-View%20Project-c8b89a?style=for-the-badge&logo=python&logoColor=0a0a0a&labelColor=1a1a1a)](https://github.com/strykey/IRIS)

A multithreaded reconnaissance tool that scans random public IPs and surfaces exposed devices in real time.<br/>
Cameras, routers, admin panels, live streams, forgotten websites.<br/>
The internet is bigger and stranger than what Google shows you. IRIS finds what is sitting in plain sight.<br/>

<br/>

[![OrbsHacker](https://img.shields.io/badge/OrbsHacker-View%20Project-c8b89a?style=for-the-badge&logo=python&logoColor=0a0a0a&labelColor=1a1a1a)](https://github.com/strykey/orbshacker)

Fakes game processes for Discord Orb quests without installing the actual games.<br/>
Because nobody has time to download 500GB just to click a button on Discord.<br/>
It works, it is stupid, and I have zero regrets building it.<br/>

</div>
